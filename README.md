MeetEaseAI is an AI-powered meeting assistant designed to streamline your workflow by transcribing meetings, generating intelligent summaries, and extracting actionable insights automatically.

## Features

- **Automated Transcription**: Convert meeting audio/video into accurate text.
- **AI Summarization**: Get concise summaries of long discussions using LLMs.
- **Action Item Extraction**: Automatically identify tasks, deadlines, and owners.
- **Searchable Archives**: Easily find past meeting notes and decisions.


### Prerequisites

- [Node.js](https://nodejs.org/) (v16+) or [Python](https://www.python.org/) (3.9+)
- API Keys for [OpenAI/Anthropic/etc.]
